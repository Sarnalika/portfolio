British Airways Customer Feedback Analysis & Predictive Modeling

Project Description

The objective of this project was two-fold: to analyze customer feedback for British Airways and to build a predictive model to forecast customer bookings. Here's how it was achieved:

  • Customer Feedback Analysis: The first task was to scrape customer reviews from Skytrax, clean the collected data, and perform sentiment analysis to gauge customer satisfaction. Additionally, topic modeling was conducted to uncover common themes and identify key factors influencing customer opinions.
  • Predictive Modeling: A predictive model was built using Python's RandomForest algorithm to forecast customer bookings based on historical data. The model's performance was evaluated through various metrics, and visualizations were created to highlight the key factors that influenced the predictions.

Objectives

The primary objectives of this project were to:

  • Scrape and clean customer feedback from Skytrax.
  • Perform sentiment analysis and topic modeling to uncover insights from customer reviews.
  • Build a predictive model using historical data to forecast customer bookings.
  • Evaluate the model's performance and identify key influencing factors.
