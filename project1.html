Weather Prediction Web App

Objective

This project aims to develop a weather prediction web app that forecasts temperature and humidity using live and historical weather data. The app provides city-specific insights, helping users make informed decisions.

Problem Statement

Accurate weather forecasting is essential for agriculture, travel, and disaster management. Traditional systems are often inaccurate or generalized for large regions. This project integrates real-time weather data with historical records to improve accuracy.

Dataset

The dataset consists of real-time weather data obtained from the OpenWeatherMap API and historical weather records. The dataset includes attributes such as temperature, humidity, wind speed, precipitation, and atmospheric pressure.

Approach

  • Utilized the OpenWeatherMap API to fetch real-time weather data.
  • Processed and cleaned the data using Python libraries such as Pandas and NumPy.
  • Trained machine learning models using Scikit-learn to predict temperature and humidity.
  • Used a RandomForest model to improve prediction accuracy.
  • Integrated predictions into a user-friendly web application.

Outcome

The app successfully predicts temperature and humidity with high accuracy. The Rain Model achieved a **Mean Squared Error (MSE) of 0.1507**, making it a reliable tool for weather forecasting.
